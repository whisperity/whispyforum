/config.php - configuration file

Usage

This file is automatically generated when the system is installed. It sets the configuration array and some other constants.

Flags

obligatory - this script is essential for the framework(?)
autoload - this script is automatically loaded by loader
nooutput - loading this script does not give direct output
autogenerated - this file is automatically generated by other script(s)
