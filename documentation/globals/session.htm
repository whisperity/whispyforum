$_SESSION - session data

Usage

The $_SESSION array stores the session data.

Flags

global - this variable is a global

Type

Array

$_SESSION
Description $_SESSION stores the session's data
Set at/in session cookie
session file in temp directory
Get/used by user and session manager

Contents

Array keys

  • username
  • pwd
  • uid
  • curr_ip
  • curr_sessid
  • log_status
  • log_bool
  • avatar_filename

  Array constants (default keys in the array which does not change value)

  • theme_name

Details

username
Description The session's username.
Relations In connection with users.username

 

pwd
Description The session's password (in MD5 hash)
Relations In connection with users.pwd

 

uid
Description The session's user's userID from database
Relations In connection with users.id

 

curr_ip
Description The session's user IP address.
Relations In connection with users.curr_ip

 

curr_sessid
Description The session's session id.
Relations In connection with users.curr_sessid

 

log_status
Description The login status of the user.

"guest" if the user is logged out
"user" if the user is logged in

 

log_bool
Description Boolean (TRUE/FALSE) value of the user's login status.

FALSE if the user is logged out
TRUE if the user is logged in
Relations In connection with users.loggedin

 

avatar_filename
Description Filename of the user's avatar.
Relations In connection with users.avatar_filename

 

theme_name
Description Name of the theme the user uses. (For guests, it's the default winky)
Value winky
