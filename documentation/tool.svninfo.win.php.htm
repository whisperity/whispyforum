/tool.svninfo.win.php - svn information tool (Windows)

Usage

This is a tool for developers. It outptus an SVN information.

Flags

tool - this script is an extension: it isn't used by the engine, however, it can require the framework

Dependencies

TortoiseSVN Windows SVN shell extension clients

Configuration

You can and sometimes have to modify the four DEFINE constants of the file. It stores the working copy's path, the TortoiseSVN root path and the two filenames.

Output

The script outputs repository information in a fancy box. The template and parsed information file is generated while executing, and those files are deleted at the end of execution.

Contents

Constants (defines)

  • WC_PATH
  • svnInfoFN
  • TSVNpath
  • tplFN

Details

WC_PATH
Description This is the path of the working copy. (You need to use chr(92) instead of \ characters to prevent unused string escaping)

Ending slash (\) must be omitted.
Example value "D:" .chr(92). "xampp" .chr(92). "htdocs"

which is really "D:\xampp\htdocs"

 

svnInfoFN
Description The name of the svn info filename (this file stores the generated output)
Example value cached.swcr-info.php

 

TSVNpath
Description The path of TortoiseSVN's main directory. (You need to use chr(92) instead of \ characters to prevent unused string escaping)

Ending slash (\) must be omitted.
Example value "C:" .chr(92). "Program Files" .chr(92). "TortoiseSVN"

which is really "C:\Program Files\TortoiseSVN"

 

tplFN
Description The name of the template file.
Example value cached.swcr-template.php
