Templates

Description

Templates are static HTML files located in the /templates folder. Template control are maintained by the template conductor class.

Layout

The basic layout of the template file is the following

Hello, {NAME}!
The current time is {TIME}, today is {DAY_NAME}

The dynamic places are the variables inside { } brackets. These variables are being replaced to their values runtime while parsing.

Putting comments into templates

The template files can have two types of comments:

Comments removed while parsing

The removed comments are placed in between <!--- and --> (first three, then two hyphens). Comments in this format will be removed from the HTML output while parsing.

Comments outputted

If you want comments to be sent to your browser (in the output), put it between <!-- and --> characters (two and two hyphens). This is also an HTML output which isn't parsed by the browser, but the template conductor does not remove it.

Example

Let's call the above template welcome (thus it's filename and path is /templates/welcome.htm). When the template is used, it's called with the following script.

$Ctemplate->useTemplate("welcome", array(
    'NAME'    =>    "John Doe",
    'TIME'    =>    "12:04:25",
    'DAY_NAME'    =>    "Tuesday"), FALSE);

It'll give the output:

Hello, John Doe!
The current time is 12:04:25, today is Tuesday.

You can always test your template without modifing the script with the template tester tool.
