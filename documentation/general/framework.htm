Engine/framework information

Introducion

The engine or framework consists of two layers: the working layer and the output layer.

Working layer

The working layer is everything that can be used to script the engine. It includes all classes, functions and globals.

Output layer

The output is in five main boxes.

Header
Left Center Right
Footer

Header

The header is generated by the loder script.

Left

The left menubar is generated by the loader script and usally consists of templates.

Center

The center box is started by the loader script, however, it's contet is generated by individual script files.

Right

The right menu box is generated by the DoFooter() function after closing the center box. This function has to be called by every individual pages.

Footer

The footer is generated by the DoFooter() function.

Example framework start-to-end call list

  • Individual webpage

    • calls loader (by inclusion)
      • loader loads template conductor
      • header generatior
      • futher class cloading and site checks
      • left menubar generation
      • begin center box
    • generates individual content
    • calls DoFooter()
      • close center box
      • generates right menubar
      • generates footer

Example individual page file (/example.framework.php)

<?php
/* Individual file... just an example */
include('includes/load.php'); // Call the loader
// Now, the center box is started, creating individual content
 $Ctemplate->useTemplate("wf_hello", array(
    'NAME'    =>    "Individual",
    'VERSION'    =>    "1.0",
    'CODENAME'    =>    "Winky"
), FALSE);
echo "<br>\n";
echo "Example wf_hello done.<br>";
 echo "You should see the framework in front of you.";
 // Generate the right box and the footer
 DoFooter();
?>
