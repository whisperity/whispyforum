Connect
Description Connect is a public function in the mySQL layer.
Connect initializes a database connection. If the connection was successful, it initializes a database selection.

If there's any error encountered while connecting, it returns with error message(s).
Sets _link
Gets configuration array
_link
Calls __giveConnectionError
__giveDBSelectError

 

TestConnection
Description TestConnection is a public function in the mySQL layer.
TestConnection does a test connection. It does not give direct error output (like how Connect() connects), and it does not select the database

It returns with a boolean variable based on the success of testing.
Called by install script
Sets _link
Gets configuration array
_link
Returns boolean value:
  • TRUE if the connection was successful
  • FALSE if there was connection errors
