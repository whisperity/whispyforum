useTemplate(templateName, replaceArray, varOutput)
Description useTemplate is a public function in the template conductor.
useTemplate orders the template conductor to read a template file, replace template variables to it's set values and echo it parsed
Parameters templateName (default NULL) - specifies the template's filename
replaceArray (default NULL-NULL array) - an array of replaceable variables and values
varOutput (default FALSE) - if FALSE, function will give echo output; if TRUE, output will be variable (return) output
Sets output
Gets output
Calls __getTemplate(templateName)
