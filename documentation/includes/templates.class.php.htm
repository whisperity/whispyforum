__getTemplate(templateName)
Description __getTemplate is a private function in the template conductor.
__getTemplate reads the template data and stores it in the _output varaible

If the template file does not exist, read a placeholder error template and stores that as output
Parameters templateName (default NULL) - specifies the template's filename
Sets _output
Gets template from the /templates directory

useTemplate(templateName, replaceArray, varOutput)
Description useTemplate is a public function in the template conductor.
useTemplate orders the template conductor to read a template file, replace template variables to it's set values and echo/return it parsed
Parameters templateName (default NULL) - specifies the template's filename
replaceArray (default NULL-NULL array) - an array of replaceable variables and values
varOutput (default FALSE) - if FALSE, function will give echo output; if TRUE, output will be variable (return) output
Sets _output
Gets _output
Calls __getTemplate(templateName)
__giveOutput

 

useStaticTemplate(templateName, varOutput)
Description useStaticTemplate is a public function in the template conductor.
useStaticTemplate orders the template conductor to read a template file and echo/return it unparsed
Parameters templateName (default NULL) - specifies the template's filename
varOutput (default FALSE) - if FALSE, function will give echo output; if TRUE, output will be variable (return) output
Calls __getTemplate(templateName)
