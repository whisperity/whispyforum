includes/(safe)load.php - framework and environment loader

Usage

This script loads every required libraries and checks vital files to load the framework.

Flags

obligatory - this script is essential for the framework

Details

The script does not have any functions, variables or classes. It does every inclusion and checking when include()-ed.

The loader loads the configuration file and the classes.

This script generates the framework as well and defines the footer-generator function.
The script can be called as safeload.php. If did so, it won't generate the frameset, it'll only load the environment.
