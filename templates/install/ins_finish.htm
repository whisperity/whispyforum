Just a few steps...
The installer done everything it could. It made a config file, installed the database and tables, and registered your root user. However, there are a few optional but recommended steps to ensure your safety.

To make sure externals cannot corrupt your configuration, first, make your configuration file (/config.php) read-only. The normal usage of the engine does not require the writing rights.

Secondly, remove the install.php file from the root (/) directory. If you do so, entering the install file's path into the browser give a 404 error.
This is the end of the installation. Have a nice upcoming year or more with engine.
